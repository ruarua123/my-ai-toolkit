# 🛠️ my-ai-toolkit

> AI 开发工具箱：plugin, skill, hook, mcp 等

[![GitHub stars](https://img.shields.io/github/stars/ruarua123/my-ai-toolkit)](https://github.com/ruarua123/my-ai-toolkit/stargazers)
[![GitHub license](https://img.shields.io/github/license/ruarua123/my-ai-toolkit)](https://github.com/ruarua123/my-ai-toolkit/blob/main/LICENSE)

## 📁 项目结构

```
my-ai-toolkit/
├── skills/          # OpenClaw Skills
├── plugins/         # 各种插件
├── hooks/           # Git hooks / CI hooks
├── mcp/             # MCP 服务器/工具
├── templates/       # 项目模板
├── scripts/         # 实用脚本
└── docs/            # 文档
```

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/ruarua123/my-ai-toolkit.git
cd my-ai-toolkit

# 查看可用的 skills
ls skills/
```

## 📦 内容说明

### skills/
OpenClaw Agent 技能包，每个子目录是一个独立的 skill。

### plugins/
各种 AI 相关插件，按功能分类存放。

### hooks/
Git hooks 和 GitHub Actions 工作流。

### mcp/
MCP (Model Context Protocol) 服务器和工具配置。

### templates/
项目模板，如 Claude Code、OpenClaw Agent 模板。

### scripts/
实用安装脚本和配置脚本。

### docs/
使用文档和贡献指南。
